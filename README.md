## Install

```bash
zylos add recall
```

Or manually:

```bash
cd ~/zylos/.claude/skills
git clone https://github.com/zylos-ai/zylos-recall.git recall
cd recall && npm install
```

## Configuration

Edit `~/zylos/components/recall/config.json`:

```json
{
  "enabled": true
}
```
